# أثير الأثر: رحلة المبعوث (Aether of Impact 3D)

لعبة جوال ثلاثية الأبعاد تعليمية سلمية، تعتمد مبدأ "التعلم الضمني" — التعلّم بالحركة والسببية والاستجابة البيئية، بلا وعظ مباشر.

## التقنيات

- React + Vite + TypeScript
- Three.js (محرّك ثلاثي الأبعاد — WebGL)
- Tailwind CSS (RTL, خط Noto Kufi Arabic)
- Framer Motion (حركات الواجهة)
- Lucide React (أيقونات)
- Ionic Capacitor (تغليف كتطبيق أندرويد + اهتزاز)
- GitHub Actions (بناء APK تلقائياً)

## فلسفة اللعبة

- **أظهر ولا تُخبر**: العالم نفسه يستجيب لقيم اللاعب
- **لا عنف**: إعمار، استكشاف، تعاطف، قرار
- **التعلم الضمني**: الأثر يُعلّم، لا الكلمات

## الميكانيكيات

### أ. إحياء الأرض (الحلقة الأساسية)
العالم جرداء رمادي بارد. مشيك قرب المناطق الذابلة يُطلق هالة المبعوث المتوهجة. صرف "جوهر النور" (طاقة) يُحيي التربة. إذا تركت منطقة مُشفى جزئياً، تتدهور ببطء — حضورك ضروري للصيانة (المسؤولية).

### ب. الزمن والأولويات
لا مؤقّتات في الواجهة. الزمن يظهر بدورة ليل/نهار وفصول متفتحة. طاقة محدودة يومياً تفرض الاختيار بين:
- إحياء وردة قريبة بسرعة (قصير المدى)
- زرع شجرة ضخمة بطيئة النمو تحمي الوادي من عواصف الرماد (طويل المدى)
- استكشاف مسارات بديلة لريّ مناطق متعددة دفعة واحدة

### ج. عدسة الاستشراف
قبل أي تغيير كبير، يُفعّل اللاعب "عدسة الاستشراف" فتتحول الشاشة إلى هولوغرام أحادي اللون يُري حالة المنطقة "بعد عشر سنين" بناءً على القرار الحالي.

### د. بوصلة اليقين
بوصلة هولوغرافية تطفو وتتوهج وتتماشى فقط عندما يسلك اللاعب وفق النواميس الطبيعية. التصرف على افتراض فاسد يكسر البوصلة ويُخفت نور المبعوث.

### هـ. الأصداء (الأرواح)
مخلوقات روحية طافية تسكن الأرض. لا قتال. الخلافات تُحلّ بألغاز حركية ومكانية (توجيه روحين ضائعين للقاء في نقطة وسطى، مشاركة مصادر النور مادياً).

## التشغيل المحلي

```bash
npm install
npm run dev
```

## إعداد Capacitor لأندرويد

```bash
npm run build
npx cap add android
npx cap sync android
npx cap open android
```

### الاهتزاز (Haptics)
- لمس خلية إحياء → نبض خفيف
- تفاعل مع صدى → نبض متوسط / إشعار نجاح
- عدسة الاستشراف → إشعار

## بنية المشروع

```
src/
  game/
    engine.ts        — محرّك اللعبة الرئيسي (Three.js)
    terrain.ts        — شيدر التضاريس (تحوّل رمادي → أخضر)
    avatar.ts         — المبعوث (الشخصية + الأثر)
    echoes.ts         — الأصداء (الأرواح)
    environment.ts    — السماء، الإضاءة، دورة الليل/النهار، الجسيمات
    types.ts          — أنواع البيانات
    GameContext.tsx   — إدارة حالة الواجهة
    haptics.ts        — واجهة الاهتزاز
  components/
    GameCanvas.tsx    — جسر Three.js + React
    Joystick.tsx      — عصا التحكم الافتراضية
    HUD.tsx           — واجهة اللعب (إحصائيات، بوصلة، أزرار)
    IntroScreen.tsx   — شاشة البداية
    EndingScreen.tsx  — شاشة النهاية
  App.tsx             — الربط الرئيسي
```
